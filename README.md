# Getting Started

**What is Initiative?**

**Initiative** is a versatile, all-purpose mod menu developed specifically for testing applications. Designed with an emphasis on simplicity and adaptability, **Initiative** features a clean and intuitive interface tailored to the needs of developers. Whether troubleshooting complex logic or rapidly prototyping new concepts, **Initiative** serves as a dependable foundation to streamline and support the development process.

# Installation

**Requirements**
- Call of Duty: Black Ops II
- Open Asset Tool
- Visual Studio Code

```bash
# Clone The Repository
git clone https://github.com/Xeirh/zm_menu.git
```

**Batch File**
- GAME_FOLDER=Your Root Black Ops II File Path
  - Example: `c:\program files (x86)\steam\steamapps\common\call of duty black ops II`
- OAT_BASE=Your Root Open Asset Tool File Path
  - Example: `c:\users\ryan\documents\openassettools`

**Workspace**

This project includes a workspace, and it's recommended to open the project through it after configuring the batch file. Using the workspace helps keep everything organized and easier to navigate. Everything not listed here is hidden in the workspace, as you don’t need to see or interact with it.

> operation: storing scripts.
> 
> source: storing the menu structure.
> 
> zone_source: linking for the building.
> 
> preload.gsc: initialization tasks.
> 
> preprocessing.gsh: header for preprocessing.
